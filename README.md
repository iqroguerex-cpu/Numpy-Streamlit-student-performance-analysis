## 🚀 Overview

A clean and interactive **Gradio web app** for generating, analyzing, and filtering student scores using **NumPy**. This project showcases practical data analysis combined with an intuitive UI.

---

## ✨ Features

* 🎲 **Random Score Generation**
* 📈 **Detailed Statistical Analysis**
* 🥇 **Top 3 Students Identification**
* 🔍 **Custom Score Filtering**
* ⚡ **Fast & Interactive UI (Gradio)**

---
