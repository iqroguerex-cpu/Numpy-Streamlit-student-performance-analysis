## 🚀 Overview

An interactive **Streamlit web application** to generate, analyze, and filter student scores using **NumPy**. This project demonstrates data analysis, state management, and UI building in Python.

---

## ✨ Features

* 🎲 Random student score generation
* 📈 Full statistical analysis
* 🏆 Top 3 performers
* 🔍 Custom score filtering
* ⚡ Fast and responsive UI with Streamlit

---

## 🛠️ Tech Stack

* Python 3.x
* Streamlit
* NumPy

## 🧠 Functionality

### 📊 Score Generation

* Creates student names dynamically
* Assigns random scores (0–100)

### 📈 Analysis

* Average, highest, lowest, total
* Pass/Fail count
* Top 3 students

### 🔍 Filtering

* Displays students scoring above a chosen threshold

## 🚀 Deployment (Streamlit Cloud)

1. Push code to GitHub
2. Go to https://streamlit.io/cloud
3. Click **New App**
4. Select repo & `app.py`
5. Deploy 🎉
